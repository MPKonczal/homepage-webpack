Tydzień 1 (15.04.2020)

Wstęp do szkolenia i zasady

Przedstawienie informacji praktycznych na temat szkolenia i określenie zasad - oglądanie lekcji, sprwdzenie prac domowych, ponowne obejrzenie lekcji przy wykonywaniu zadań i robienie notatek. Poza nauką na szkoleniu można szlifować swój angielski.

Czym jest frontend?

Frontend odpowiada za wygląd stron / aplikacji internetowych oraz umożliwia na interkację z użytkownikiem. Jest generowany głównie przy użyciu trzech technologii: HTML, CSS i JavaScript. Technologie frontendowe przejmują rynek tradycyjnych aplikacji w wybranych obszarach.

Narzędzia i grupy

Szkolenie opiera się na następujących narzędziach: przeglądarka Google Chrome i oprogramowanie Visual Studio Code. Głównym kanałem komunikacji jest Slack, a poza tym istnieje również grupa na Facebooku.

Ścieżka rozwoju

W branży IT można wyróżnić następujące stopnie doświadczenia: Stażysta / Junior, Mid, Senior, Expert / Ninja / Wizard.
Podczas nauki należy wziąć pod uwagę efekt Krugera-Dunninga, czyli zależność pomiędzy pewnością siebie a doświadczeniem.
Tutaj jest dostępna roadmapa dla Frontend Developera.

Konfiguracja MacOS / Windows / Linux

Nie ma znaczących różnic w pracy na wymienionych systemach operacyjnych. Nazwa katalogu roboczego nosi nazwę dev, a każdy projekt należy otworzyć za pomocą Visual Studio Code (VSC -> File -> Open Folder).

Tydzień 2 (20.04.2020)

Warstwy strony internetowej

Strony internetowe posiadają następujące warstwy:
treść (warstwa 0),
semantyka - HTML (warstwa 1),
prezentacja - CSS (warstwa 2),
interakcja - JavaScript (warstwa 3).
Należy pamiętać, że każda z tych warstw ma swoją odpowiedzialność oraz content is king!

HTML - wstęp

HTML = HyperText Markup Language (hipertekstowy język znaczników)
Odpowiada za treść strony internetowej, a szczególnie za jej właściwe opakowanie. Treścią są teksty, nagłówki, linki, listy, definicje, obrazki, video, formularze, wszystko co jest nośnikiem informacji. HTML składa się ze znaczników / tagów /elementów i to one są analizowane przez przeglądarkę. Budowa znacznika:
<znacznik atrybut="wartość">treść</znacznik>.
Na stronach internetowych należy stosować semantyczny kod, czyli taki, który ma swoje znaczenie. Pliki zawierające kod HTML mają zwykle rozszerzenie .html. Główne znaczniki, które powinny być wykorzystywane przez frontendowca to:
<main>,
<section>,
<article>,
<header>,
<footer>,
<h1 - h6>,
<p>.
Każda sekcja powinna mieć nagłówek. Po napisaniu kodu HTML zawsze należy zweryfikować jego poprawność w walidatorze.
HTML ma następujące cechy: opakowuje i opisuje treść strony, mówi przeglądarce o strukturze dokumentu, posiada swoją zdefiniowaną składnię, nie jest językiem programowania, jest językiem znaczników, aktualną rekomendowaną wersją jest 5.2, a całą specyfikację można znaleźć na stronie w3c.

CSS - wstęp

CSS = Cascading Style Sheets (kaskadowe arkusze stylów)
Odpowiada za prezentację elementów HTML na ekranie, wydrukach, itp. Składnia CSS to reguły. Budowa reguły CSS:
selektor {
właściwość: wartość;
}.
Pliki zawierające kod CSS mają rozszerzenie .css. Istnieje również możliwość korzystania z reguł CSS bez używania zewnętrznych plików (niezalecana). Zatem można stylować poprzez znacznik <link> lub znacznik <style> w sekcji <head>.
Pierwszą metodą stylowania jest stylowanie po elementach. Właściwości i wartości zapisujemy po angielsku, a w przypadku wartości dodatkowo można posłużyć się kodem szesnastkowym (dla kanałów RGB).
CSS ma następujące cechy: odpowiada za prezentację strony, nie ma jednej wersji (rozwój modułów), potocznie mówi się, że aktualna wersja to 3. Przeglądarki potrafią różnie interpretować CSS co prowadzi do tego, że pisanie stylów powoduje wiele frustracji u programistów, ale jest też niezłym wyzwaniem. Całą specyfikację można znaleźć na stronie w3c.

Magia programowania

Programowanie można porównać do gotowania z tą różnicą, że przepisy są często wymyślane na bieżąco. Algorytm jest sposobem zapisu kroków, które powinny zostać wykonane w odpowiedniej kolejności. Zazwyczaj są one zapisywane w postaci schematu blokowego. Jako programowanie można uznać zamianę ludzkiego języka na język zrozumiały przez komputer. Każdy jezyk programowania ma swoją składnię, gramatykę i ortografię. Na świecie jest ponad 700 języków programowania, natomiast w szkoleniu poznawany jest jeden - JavaScript.
Jest to najbardziej potężna część frontendu. Może modyfikować style (CSS), HTML i treść, zatem potrafi wpływać na każdą z warstw strony internetowej. Odpowiada za interakcje, wywoływanie innych akcji ponad te standardowe, rozszerzając możliwości zwykłych stron. Aktualnie jest jednym z najpopularniejszych języków programowania na świecie.

DevTools: podglądanie i zmienianie istniejących stron

Narzędzia deweloperskie w przeglądarce Google Chrome można uruchomić, wybierając dowolny element strony internetowej i klikając prawy przycisk myszy, a następnie Zbadaj (Inspect) lub poprzez naciśnięcie klawisza F12. Dzięki temu można podejrzeć treść, HTML oraz style. Wszelkie wprowadzone zmiany na stronie są widocznie od razu, ale tylko lokalnie ze względu na brak dostępu do plików źródłowych (zmiany znikają w momencie odświeżenia strony). DevTools są przydatne po to, aby móc szybko przetestować co można ewentualnie zmienić na stronie, a później to wprowadzić do edytora kodu.
